# NIFTY 100 Financial Analytics Dashboard

## Skills
Python, SQL, Power BI

## Project Overview
Analyzed financial data of NIFTY 100 companies to evaluate business performance, profitability, assets, liabilities, and cash flow trends.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Financial KPI Generation
- Business Insights Extraction
- SQL-Based Financial Analysis
- Interactive Power BI Dashboard

## KPIs
- Net Profit Margin
- Debt-to-Equity Ratio
- Free Cash Flow
- Total Sales
- Total Profit
- Total Assets

## Technology Stack
- Python
- Pandas
- NumPy
- SQLite
- Power BI

## Project Workflow
1. Data Understanding
2. Data Cleaning
3. Exploratory Data Analysis
4. Business Insights
5. SQL Analysis
6. Power BI Dashboard

## Outcome
Built an end-to-end financial analytics solution that provides insights into the financial health and performance of NIFTY 100 companies through interactive visualizations and data-driven analysis.
